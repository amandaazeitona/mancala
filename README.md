# Mancala

A Mancala game implementation developed for the **Data Structures II** course in the Bachelor's degree in Computer Science at the University of Brasília (UnB).

The project features both **Player vs Player** and **Player vs Computer** modes, with an AI based on the **Minimax** algorithm.

## Running the Project

Compile the project:

```bash
gcc -o kalah kalah.c
```

Run:

```bash
./kalah
```
